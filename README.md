# Vector Data Processor 📊

Este projeto consiste em um script Python desenvolvido para realizar **leitura, processamento e persistência de dados vetoriais** a partir de arquivos de texto.

## 🚀 Funcionalidades

- **Extração de Dados:** Leitura automatizada de arquivos brutos (`vetor_a.txt`, `vetor_b.txt`).
- **Processamento:** Manipulação de listas e estruturas de dados em memória.
- **Persistência:** Exportação dos resultados processados para arquivo de saída (`vetor_c.txt`).

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3
- **IDE:** PyCharm
- **Conceitos:** Manipulação de Arquivos (I/O), Lógica de Programação, Estruturas de Dados.

## 📂 Estrutura do Projeto

```bash
├── main.py        # Script principal de execução
├── vetor_a.txt    # Input de dados A
├── vetor_b.txt    # Input de dados B
└── vetor_c.txt    # Output gerado pelo script
```

## 🎯 Objetivo de Aprendizado

Demonstrar competência na manipulação de arquivos e lógica de programação backend, simulando um pipeline simples de **ETL (Extract, Transform, Load)**.
